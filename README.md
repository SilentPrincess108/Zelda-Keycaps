# Zelda-Keycaps

A CAD project made for Hack Club's Stasis! For this project, I created 4 custom Zelda-themed keycaps(made to fit MX switches) for my pathfinder project! They feature a triforce, navi, a rupee, and a heart container! I also created this project as an easy and simple start into learning Fusion, as I'm still pretty new to 3d modeling!
